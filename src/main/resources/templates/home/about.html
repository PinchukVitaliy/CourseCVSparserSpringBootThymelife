About the project

The product was created to work on CSV-format files. Any user has the right to register and work on

the contents of the file from any place where there is Internet access. Since this is a web site, access to it

possible from anywhere in the world. The site contains statistics of each created or downloaded file,

namely, what was done on this file. Ability to disable users and assignment of certain rights (roles).

Features of ordinary users:

  • upload existing file
  • modify file contents
  • create new file
  • download the file itself
  • give editing access, viewing or downloading a file to another user

Administration features:

  • setting roles
  • lock and unlock user
  • view and clear all statistics for all files
